# Section Editing Rules

- **Scope**: One section or one coherent change at a time (e.g. one paragraph, one subsection). Prefer small, reviewable edits.
- **LaTeX**: Edit only under `latex/sections/` (or `latex/main.tex` for structure). Preserve existing macros and bibliography style.
- **Rewrite patterns**: (1) State the point in one sentence, then expand; (2) One claim per sentence where possible; (3) After adding a claim, add or verify the citation/evidence in the same edit.
- **Evidence**: Before writing a claim (e.g. "X improves over Y"), confirm the evidence exists (result file, config, table). If not, either remove the claim or mark it as "[TODO: add result]".
- **Summary**: After each edit pass, list what was changed and which file(s); note any claims that still need evidence.
