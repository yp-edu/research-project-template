# Review & Revision Skill

## When to Use

- The user is preparing for submission or an internal milestone.
- The user wants a systematic review of methodology, experiments, or writing.
- You need to assess reproducibility and identify missing evidence.

## Instructions

- Use `references/reproducibility-checklist.md` to assess whether experiments can be reliably reproduced.
- Use `references/internal-review-checklist.md` to structure a reviewer-style pass.

When running a review:

1. Ask which aspects to focus on (methods, experiments, writing, reproducibility, or all).
2. Walk through the relevant checklist(s) and record issues by severity.
3. For each issue, suggest concrete next steps and which subagent or command to use.
4. Provide a brief readiness assessment and prioritized follow-up list.
