# Literature Review Skill

## When to Use

- The user is exploring a new topic or refining the problem statement.
- The user wants to map existing baselines, SOTA methods, or related work.
- The user needs help organizing notes for background or related work sections.

## Instructions

- Use `references/note-structure.md` to structure notes for each paper.
- When proposing searches, suggest concrete query strings and venues.

When running a literature pass:

1. Ask the user for the precise topic, constraints, and any seed papers.
2. Propose a small set of search directions (e.g., venues, keywords, time windows).
3. For each important paper, capture the elements in the note structure file.
4. Group papers into themes or baselines that can inform experiments and writing.
