# Research Question Guidance

- Question notes capture uncertainties, directions, limitations, low-hanging fruits, scientific locks, explored paths, and abandoned paths.
- Vague ideas, intuitions, and hunches should become testable questions with explicit assumptions, expected evidence, metrics, and decision rules when possible.
- Each active question should point to the claim, decision, or experiment that can resolve it.
- Closed questions should contain or link to the answering claim, deferring decision, cancelling decision, or experiment conclusion.
- Keep category-level summaries in `README.md` factual; leave it title-only until there is something real to summarize.
- Track question files from the category dashboard as `[question](question.md): Pending`, then replace `Pending` with the answering claim, deferring decision, cancelling decision, or linked experiment conclusion.
