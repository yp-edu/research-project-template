# Docs Bootstrap Template

Create or update these files:

1. **`docs/README.md`**
   - Snapshot
   - Research focus
   - Key artifacts
   - Memory index
   - Next steps
   - Open questions / decisions
2. **`docs/questions/README.md`**
   - Problem framing
   - Primary questions
   - Hypotheses
   - Evaluation criteria
3. **`docs/literature/README.md`**
   - Search focus
   - Seed papers
   - Themes and gaps
   - Notes index
4. **`docs/experiments/README.md`**
   - Active tracks
   - Baselines
   - Planned runs
   - Result index
   - Decisions and lessons

Keep the memory short and editable; the goal is durable project context, not a second copy of the paper.
